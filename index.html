Compiler-Ready Source Note Audit

The copy-paste exports contain 12,298 document-level FRUS Source notes and zero style or extraction-safety failures. Each exported row begins with exactly one leading Source: label and excludes editorial-note, empty-stub, and pending-placeholder status rows.

Two notes retain a second internal Source: string because the published FRUS note itself contains that text; five notes retain official redacted source-path forms such as Source: [ ... not declassified.
